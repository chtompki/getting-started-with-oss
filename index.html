Figuring out how to get involved.

Now that you have an idea what you want to do, how do you do it?

  • Reading and more reading (of the project's website).
  • Website Link: "contributing," "contributors," "community," ect.
  • Checkout the code, run unit tests.
  • Find a small bug, or improve unit tests.

Whom do I first contact with questions?

  1. Email list.
  2. Github contexts: /commits/master and /graphs/contributors
  3. Github "blame" view on a specific file: /blame/master/README.md

Making your first commits.

  • Personally, I'm always excited to see anyone contribute.
  • Getting used to a code base takes time, and helping others contribute makes your job easier.
  • Unit tests are like brushing your teeth, and use the code style around you

Staying involved.

  • Participate in discussion.
  • Find a single portion of the code base you can master.
  • Ask senior members where the best place to help is.
  • Volunteer for things others might not want to do, and do them.

What I've been working on for the last few months in the Apache Commons Project.

Anyone not familiar with the Apache Commons Project?

Introducing commons-text.

Goal. Text processing algorithms out of standard java library scope.

Currently.

  • String Escaping (e.g. XML, CSV, JSON, ect.)
  • String similarity and edit distance.
  • A String diff tool based on the LCS Algorithm.

String Similarity Example

The Longest commons subsequence is a classical String similarity algorithm.

LongestCommonSubsequence siml = new LongestCommonsSubsequence();
siml.apply("abba","abab"); // 3
siml.apply("frog", "fog"); // 3
siml.apply("PENNSYLVANIA", "PENNCISYLVNIA") //11

LCS Edit Distance

LongestCommonSubsequenceDistance dist = new LongestCommonSubsequenceDistance();
dist.apply("abba","abab"); // 2
dist.apply("frog", "fog"); // 1
dist.apply("PENNSYLVANIA", "PENNCISYLVNIA") // 3
dist.apply("elephant", "hippo")// 11

Levenshtein Edit Distance

Different algorithm same results.

LevenshteinDistance dist = new LevenshteinDistance();
dist.apply("abba","abab"); // 2
dist.apply("frog", "fog"); // 1
dist.apply("PENNSYLVANIA", "PENNCISYLVNIA") // 3
dist.apply("elephant", "hippo")// 7
